Thanks for using my app !

This application starts the music player when a bluetooth device connects.
You can select which music player starts for a specific bluetooth device.

Following permissions are required:

  • BLUETOOTH
    to watch for bluetooth devices connexions

  • BLUETOOTH_ADMIN
    to disable bluetooth (if the setting is enabled)

  • WRITE_EXTERNAL_STORAGE
    to save application preferences

If you encounter some issues, please feel free to send me details on my email present on the PlayStore.
