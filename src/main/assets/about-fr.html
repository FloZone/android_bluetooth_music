Merci d'utiliser mon application !

Cette application démarre la musique quand un appareil bluetooth se connecte.
Vous pouvez choisir le lecteur de musique à lancer pour chaque appareil bluetooth.

Les permissions suivantes sont nécessaires au fonctionnement de l'application:

  • BLUETOOTH
    pour détecter les connexions bluetooth

  • BLUETOOTH_ADMIN
    pour désactiver le bluetooth (si l'option est activée)

  • WRITE_EXTERNAL_STORAGE
    pour sauvegarder les préférences de l'application

Si vous rencontrez des problèmes, merci de m'envoyer les détails sur mon email présent sur le PlayStore.
